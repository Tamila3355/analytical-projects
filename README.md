Sales Dashboard – Power BI Project
Cel projektu:
Pokazanie umiejętności analizy danych, modelowania oraz tworzenia wizualizacji w Power BI na podstawie syntetycznego zbioru danych.
________________________________________
Zawartość repozytorium
•	sales_data.csv – syntetyczny zbiór danych sprzedażowych
•	dashboard_screenshots/ – zrzuty ekranu dashboardu Power BI
• Sales_Dashboard.pbix – plik raportu Power BI
________________________________________
Użyte technologie
•	Power BI
•	Power Query
•	DAX
•	Excel (przygotowanie danych)
________________________________________
Opis projektu
Projekt przedstawia analizę sprzedaży na podstawie wygenerowanego zbioru danych, który odzwierciedla typową strukturę biznesową:
•	data transakcji
•	kategoria produktu
•	cena
•	ilość
•	wartość sprzedaży (Total)
W raporcie Power BI znajdują się:
•	analiza sprzedaży według kategorii
•	dzienna dynamika sprzedaży
•	karty KPI (Total Sales, Average Price, Total Quantity)
________________________________________
Cel analityczny
Celem było zaprezentowanie umiejętności:
•	przygotowania i czyszczenia danych
•	tworzenia miar DAX
•	budowy czytelnego dashboardu
•	analitycznego myślenia i prezentacji danych
